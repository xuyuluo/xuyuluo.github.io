How to keep processes running after ending ssh session

The first thing you need to do is to install screen

Execute screen in terminal, then you get into a virtual terminal. You are supposed to run you program that you want it keep running in the virtual terminal.

Then, 1: pressCtrl+A, 2: pressD, to detach current virtual terminal and bring you back to the normal terminal. Use screen -r take you back to virtual terminal.

Failed to import the numpy c-extensions under conda env

Reinstall, numpy package

  1. activate conda env
  2. pip uninstall -y numpy
  3. pip uninstall -y setuptools
  4. pip install setuptools
  5. pip install numpy
